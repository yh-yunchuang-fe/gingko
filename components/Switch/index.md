在两个互斥对象进行选择，eg：选择开或关。

## API

| 属性          | 说明       | 类型     | 默认值  |
| ------------- | ---------- | -------- | ------- |
| width         | 开关的宽度 | number   | 48      |
| height        | 开关的高度 | number   | 28      |
| size          | 按钮大小   | number   | 18      |
| color         | 主题颜色   | string   | #FE8F1D |
| bgColor       | 背景颜色   | string   | #C4C4C4 |
| btnColor      | 按钮颜色   | string   | #FFFFFF |
| onValueChange | 点击事件   | function | (){}    |
