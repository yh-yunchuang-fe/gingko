## API

| 属性         | 说明         | 类型   | 默认值 |
| ------------ | ------------ | ------ | ------ |
| labelSty     | 左侧文字样式 | object | {}     |
| valueSty     | 右侧文字样式 | object | {}     |
| labelWarpSty | 左侧容器样式 | object | {}     |
| valueWarpSty | 右侧容器样式 | object | {}     |
| label        | 左侧内容     | string | ''     |
| value        | 右边内容     | string | null   |
