可配置多位短信验证码组件

## API

| 属性         | 说明             | 类型     | 默认值 |
| ------------ | ---------------- | -------- | ------ |
| defaultValue | 默认值           | string   | --     |
| size         | 验证码位数       | number   | 6      |
| autoFocus    | 是否自动获取焦点 | boolean  | false  |
| onChange     | 状态变更事件     | function | (){}   |
| onFocus      | 获取焦点事件     | function | (){}   |
| onBlur       | 获取焦点事件     | function | (){}   |
