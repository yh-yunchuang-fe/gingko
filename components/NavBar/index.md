头部导航条

## API

| 属性         | 说明         | 类型         | 默认值 |
| ------------ | ------------ | ------------ | ------ |
| style        | 容器样式     | Object       | {}     |
| leftIcon     | 左侧图标     | string       | back   |
| leftContent  | 左侧内容     | ReactElement | null   |
| rightContent | 右侧内容     | ReactElement | null   |
| onLeftClick  | 左侧点击事件 | function     | (){}   |
