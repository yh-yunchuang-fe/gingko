位于 APP 底部，方便用户在不同功能模块之间进行快速切换。

## API

| 属性           | 说明      | 类型     | 默认值    |
| -------------- | --------- | -------- | --------- |
| filter         | 当前下标  | number   | 0         |
| iconStyle      | icon 样式 | object   | {}        |
| onClick        | 点击事件  | function | (index){} |
| tabTextStyle   | 文字样式  | object   | {}        |
| selectionColor | 选中颜色  | string   | #FE8F1D   |
| tabs           | 数据      | Array    | []        |
