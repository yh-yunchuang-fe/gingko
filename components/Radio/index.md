单选框和单选组

## API

| 属性           | 说明             | 类型                  | 默认值 |
| -------------- | ---------------- | --------------------- | ------ |
| icon           | 多选框图标       | `boolean`\|`function` | true   |
| dir            | 书写方向         | `left`\|`right`       | left   |
| checked        | 是否选中         | boolean               | false  |
| defaultChecked | 默认是否选中     | boolean               | false  |
| onChange       | 选中状态变更事件 | function              | (){}   |
| style          | 容器样式         | Object                | {}     |
| textStyle      | 文本样式         | Object                | {}     |
| disabled       | 是否禁用         | boolean               | false  |
