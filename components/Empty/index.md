在整张页面中组织插画、图标、文字等内容，向用户反馈操作结果。

## API

| 属性         | 说明             | 类型                  | 默认值                  |
| ------------ | ---------------- | --------------------- | ----------------------- |
| source       | Empty 返回的图片 | --                    | `require(你的图片路径)` |
| message      | Empty 返回的文字 | `string`\|`ReactNode` | 暂无数据                |
| btn          | 按钮文字         | `string`\|`ReactNode` | --                      |
| style        | 样式             | object                | {}                      |
| imgStyle     | 图片样式         | object                | {}                      |
| messageStyle | 文本样式         | object                | {}                      |
