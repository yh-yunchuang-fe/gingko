用于让用户在不同的视图中进行切换。

## API

| 属性     | 说明         | 类型                                      | 默认值  |
| -------- | ------------ | ----------------------------------------- | ------- |
| type     | 阴影类型     | `default` \| `large` \| `small` \| string | default |
| children | 内容区域元素 | ReactNode                                 | --      |
